To tidy up the tb.csv run the tidy_tb.ipynb file with any compiler which supports this extension and change the tidy.csv path to yours. The analized file is a spreadsheet file that contains formulae that summarize the statistics. The Original Data folder contains the raw file and in the Metadata folder you can find the Metadata Guide which explains how the tb.csv is structured. The Analysis Data folder contains the cleaned up version of tb.csv and the spreadsheet with its analysis. The Command files folder provides the code which cleans the tb.csv file. The Documents folder contains the readme file and the Data Appendix whichc contains additional information. about the clean tuberculosis patient data. It can be run with any .ipynb compiler.
